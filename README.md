# Dự án WordPress Cá nhân

## Mô tả
Đây là một **dự án WordPress cá nhân**, được thực hiện trong quá trình học và thực hành lập trình web.  
Dự án được thiết lập bằng **Docker** để thuận tiện cho phát triển và triển khai cục bộ.

---

## Tính năng
- Môi trường WordPress hoạt động đầy đủ  
- Chạy thông qua **Docker container**  
- Dễ dàng thiết lập môi trường phát triển cục bộ  
- Phù hợp để luyện tập phát triển **theme/plugin WordPress**  
- Bao gồm container **MySQL** làm cơ sở dữ liệu  

---

## Công nghệ sử dụng
- **WordPress** – CMS để xây dựng website  
- **Docker & Docker Compose** – Container hóa môi trường  
- **MySQL** – Cơ sở dữ liệu cho WordPress  
- **PHP** – Ngôn ngữ lập trình backend  
- **HTML / CSS / JavaScript** – Tùy chỉnh giao diện frontend  

---

## Hướng dẫn cài đặt

### 1. Clone repository
```bash
git clone https://github.com/vukha12/chuyendecms.git
cd chuyendecms
```
### 2. Chạy WordPress bằng Docker Compose
```bash
docker-compose up -d
```
### 3. Truy cập website
```bash
http://localhost:8080
```
